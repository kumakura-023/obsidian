# 演算子オーバーロードとは

以下は、金額を扱うクラスで、フィールドに金額と単位を持つ。

```C#
public struct CurrencyAmount
{
    private decimal amount;
    private string currency;

    public CurrencyAmount(decimal amount, string currency)
    {
        this.amount = amount;
        this.currency = currency;
    }
}
```

このクラスのインスタンスAとBがあるとして、それらを直接比較するとコンパイルエラーになる。

そこで、演算子オーバーロードを実装する。
```C#
    public static bool operator ==(CurrencyAmount a, CurrencyAmount b)
    {
        if (a.currency != b.currency)
            throw new ArgumentException("");

        return a.amount == b.amount;
    }
    public static bool operator !=(CurrencyAmount a, CurrencyAmount b)
    {
        if (a.currency != b.currency)
            throw new ArgumentException("");

        return a.amount != b.amount;
    }
	//==演算子のオーバーロードを行うときは、以下も実装するのが通例
    public override bool Equals(object obj)
        => obj is CurrencyAmount other && this == other;

    public override int GetHashCode()
        => HashCode.Combine(amount, currency);
```

## キャスト演算子オーバーロード

```C#
CurrencyAmount amountA = new CurrencyAmount(55.5m, "HD");
decimal d = amountA;
// d == 55.5m  → true
```

これを満たすにはどうしたらいいか。


```C#
 public static implicit operator decimal(CurrencyAmount a)
        => a.amount;
```
