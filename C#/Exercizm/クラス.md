## フィールドとプロパティの違い

### ◆ 実例で理解する（めちゃ分かりやすいやつ）

***フィールドだけの場合：***
```C#
class Character {     public int hp;   // フィールド }`
```

外部からこうアクセスできる：
```C#
c.hp = -1000;   // ← めちゃくちゃな値を入れられる`
```

完全に無防備。

***プロパティの場合：***
```C#
class Character
{
    private int _hp;  // フィールド（隠す）

    public int HP      // プロパティ（窓口）
    {
        get => _hp;
        set => _hp = Math.Max(0, value);  // ここで制御できる
    }
}

```

外から見るとこう使える：
```C#
c.HP = -1000;  // OKだけど、内部では0に補正される`
```

**プロパティを使うことで、「中身の値は守りたいけど、アクセスは許したい」という矛盾を解決できる。**

### ◆ 本質的な違いを一言で

**フィールド＝値そのものを持つ場所**

**プロパティ＝その値を“どうアクセスするか”をコントロールする仕組み**
